# Win Go Game - KE, 5D, TRX

एक complete lottery prediction game जो Android और iOS दोनों में Acode से run हो सकता है। 🎮

## 🎮 Features

### तीन अलग-अलग Games:

#### 1. **KE Game** (Kenya Lottery)
- 0-99 के बीच एक number select करो
- Winning number randomly generate होता है
- Win करो तो 2x winnings मिलेगी

#### 2. **5D Game** (5-Digit Lottery)
- 5 digit का number enter करो (00000-99999)
- Accurate prediction पर 10x winnings
- Higher stakes, higher rewards

#### 3. **TRX Game** (Crypto Betting)
- Red, Blue या Green color select करो
- Crypto betting सिस्टम
- 2x winnings on correct prediction

### अन्य Features:
- ✅ User Registration & Login
- ✅ Wallet System with Balance
- ✅ Real-time Game Timers (60 seconds per round)
- ✅ Betting History (Last 50 bets)
- ✅ Local Data Storage (localStorage)
- ✅ Responsive Mobile Design
- ✅ Beautiful UI with Animations

## 📱 कैसे Use करें Acode में?

### Step 1: Acode Download करो
- Google Play Store से "Acode" download करो
- Open करो

### Step 2: Files बनाओ

Acode में ये 3 files बनाओ:

#### File 1: `index.html`
- Acode में New File → `index.html`
- नीचे दिया गया code copy करो

#### File 2: `style.css`
- Acode में New File → `style.css`
- Style code copy करो

#### File 3: `script.js`
- Acode में New File → `script.js`
- JavaScript code copy करो

### Step 3: Project Structure बनाओ

```
win-go-game/
├── index.html
├── style.css
└── script.js
```

### Step 4: Run करो

#### Option A: Browser से (सबसे आसान)
1. Acode में `index.html` open करो
2. ऊपर right में **"Preview"** या **"Open in Browser"** button दबाओ
3. Game खुल जाएगा!

#### Option B: Live Server से
1. Acode में right menu में **"Run"** दबाओ
2. Live server start होगा
3. Browser में `http://localhost:8080` खोलो

## 🎯 How to Play

### पहला Step - Registration
1. अपना **Username** enter करो
2. अपना **Email** enter करो
3. **Login/Register** button दबाओ
4. Start करो खेलना!

### खेलने का तरीका:

#### KE Game:
```
1. Game selector से "KE" select करो
2. 0-99 के बीच कोई भी number click करो
3. Bet amount enter करो (default: $10)
4. "Place Bet" दबाओ
5. Result देखो!
```

#### 5D Game:
```
1. "5D" select करो
2. 5 digit number enter करो (e.g., 12345)
3. Bet amount enter करो
4. "Place Bet" दबाओ
5. Result आएगा!
```

#### TRX Game:
```
1. "TRX" select करो
2. Red, Blue या Green में से कोई color choose करो
3. Bet amount enter करो
4. "Place Bet" दबाओ
5. Color match होगा तो जीत गए!
```

## 🐛 Troubleshooting

### Problem: Game नहीं खुल रहा है
**Solution:**
1. Acode को restart करो
2. सभी 3 files सही folder में हैं check करो
3. File names exact same हों: `index.html`, `style.css`, `script.js`

### Problem: Data lost हो गया
**Solution:**
- Clear Cache मत करो
- Browser cache clear होने से data delete हो सकता है
- Data backup के लिए screenshot लो

### Problem: Buttons काम नहीं कर रहे
**Solution:**
1. JavaScript enable है check करो
2. Browser restart करो
3. Acode update करो
